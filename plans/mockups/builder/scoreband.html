How this connects.
Each row is one ScoreBand (Label, MinScore, MaxScore, Severity, ColorHex, IsTerminal) under this ScoreBandSet. A model references the set via ScoreModel.BandSetID; at recompute the engine assigns each score the band whose [MinScore, MaxScore] contains it. A model can't publish until its band set has at least one band.
