How to scale this signal

Raw counts aren't comparable across signals (e.g. 50 opens vs 2 transactions). rescaled values are mapped to 0–1 points before weighting. Choose how:

Scale (Min-Max)

Positions scores proportionally between lowest and highest in the group. Recommended.

Rank (Percentile)

Ranks members relative to the population. Extremely robust to outliers.

Standardize (Z-Score)

Measures standard deviation from average. Great for normal distributions.

Raw Value

Skips normalization and uses the raw number directly in calculation.
